# **App Name**: Clandestine Access

## Core Features:

- Age Verification: Premium age gate to verify users are over 18 before granting access to restricted content.
- Loading Sequence: A stylized loading screen with animated texts to create anticipation before redirecting to the sales page.
- Premium Sales Page: A high-quality sales page with a premium dark aesthetic to present exclusive content.
- Real-time Countdown Timer: Display a countdown timer that restarts on every page reload to create a sense of urgency.
- Event Tracking: Record user interactions using logEvent to monitor the conversion rates at various steps.

## Style Guidelines:

- Background: Absolute black (#000000) for a premium dark aesthetic.
- Primary color: Neon red (#ff0000) for highlights, buttons, and alerts.
- Accent color: Neon green (#3aff4f) for timer and subtle accents.
- Font: 'Space Grotesk' sans-serif for titles and 'Inter' sans-serif for body text. Note: currently only Google Fonts are supported.
- Minimalist icons in white for a clean and uncluttered design.
- Generous use of white space and padding to create a spacious, premium feel.
- Framer Motion for smooth transitions, button hovers, and loader animations.
